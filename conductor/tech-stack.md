# Tech Stack: SwiftSeat Unified Platform

## Frontend
- **Framework**: React 18+
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Custom tokens, Glassmorphism)
- **State Management**: React Context (Shared state) + Hooks
- **Icons**: Lucide React / Google Material Symbols

## Backend Integration
- **Server**: Flask (Python 3.10+)
- **API Model**: RESTful
- **AI Services**: Google Gemini 2.0 Flash
- **Data Persistence**: Firestore (Optional / Mock mode)
- **Mapping**: Google Maps Compute Route Matrix API

## Development Environment
- **Runtime**: Node.js 20+
- **Deployment**: Vercel (Frontend) / GCP (Backend)
- **Monitoring**: server.py logs / Browser console
