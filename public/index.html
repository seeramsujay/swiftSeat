How does PALO work?

The PALO algorithm (Predictive Arrival-time, Load-balanced Optimization) scores each concession stand using three factors:

🚶
Walk Time

Your real walking distance to each stand.

⏱️
Predicted Wait

Queue wait when you arrive, factoring current trend direction.

⚖️
Load Penalty

Avoids routing everyone to the same stand (prevents herd effect).

score = walk_time + predicted_wait + load_penalty

Lower score = faster total experience. ✓
